# 平面图功能优化计划

## 目标
优化平面图编辑功能，使其更易用、更美观

## 优化项

### 1. 拖拽优化
- [ ] 添加拖拽时的视觉反馈（边框高亮、透明度变化）
- [ ] 添加拖拽时的辅助线
- [ ] 改善拖拽灵敏度

### 2. 尺寸调整
- [ ] 添加调整大小的手柄（右下角）
- [ ] 拖动边角调整宽高
- [ ] 保持比例缩放选项

### 3. 自动对齐
- [ ] 网格对齐（20px网格）
- [ ] 智能对齐到其他元素边缘
- [ ] 对齐辅助线显示

### 4. 视觉美化
- [ ] 改善房间样式（更像室内设计图）
- [ ] 添加地板纹理/图案
- [ ] 添加门、窗等元素
- [ ] 优化颜色方案

## 技术实现

### 拖拽+调整大小
- 使用 resize 属性或自定义手柄
- 添加 mousedown/mousemove/mouseup 事件监听
- 使用 transform 或 left/top 进行移动

### 自动对齐
- 添加网格吸附（Math.round(x/gridSize) * gridSize）
- 添加碰撞检测和边缘对齐
- 拖拽时显示对齐辅助线

### 视觉效果
- CSS 渐变模拟地板
- 边框样式模拟墙体
- 添加门、窗等 SVG 元素
